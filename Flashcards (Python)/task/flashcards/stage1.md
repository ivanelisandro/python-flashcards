# Stage 1:

Focus on stage 1 is just creating a card with user input.
Two inputs:

- first represents the "term" (front)
- second represents the "definition" (back) 

## Examples:

Example 1.

```
Card:
> purchase
purchase
Definition:
> buy
buy
```

Example 2.

```
Card:
> cos'(x)
cos'(x)
Definition:
> -sin(x)
-sin(x)
```
